# Manual de Uso – mohamed-projeto

## Instalação

### Backend (Spring Boot)
1. Necessário Java 17+ e Gradle instalado.
2. Execute:
   ```
   cd backend
   ./gradlew bootRun
   ```

### Frontend (JavaFX)
1. Necessário Java 17+ e JavaFX SDK.
2. Execute:
   ```
   cd frontend
   ./gradlew run
   ```
3. Para Android: abra `frontend/` no Android Studio.

## Como usar
1. Abra o frontend.
2. Faça login/cadastro.
3. Navegue pelas funcionalidades principais.
4. Consulte o vídeo tutorial (link no README).
